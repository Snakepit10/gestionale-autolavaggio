Immagini brand MasterWash
=========================

Per attivare il logo nelle pagine cliente, salvare qui:

  masterwash-logo.png

Specifiche consigliate:
- Formato: PNG con sfondo trasparente
- Altezza: 200 px (la pagina lo ridimensiona)
- Larghezza: proporzionale (logo orizzontale tipico ~ 600x150)

Senza il file, le pagine mostrano un fallback testuale
stilizzato "MasterWash" in cyan italic (definito in CSS
.brand-text di templates/clients/base_public.html).

Quando aggiungi il file, fai:
  git add static/img/masterwash-logo.png
  git commit -m "Aggiunto logo MasterWash"
  git push
